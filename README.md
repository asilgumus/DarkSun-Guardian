# DarkSun Guardian

DarkSun Guardian is a bot project built with Node.js. It includes command handlers, event listeners, and utility functions to manage and automate various tasks.

## Project Structure

- `commands/` - Contains bot commands organized in subdirectories.
- `events/` - Contains event handler scripts for bot events.
- `utils/` - Utility functions used across the project.
- `app.js` - The main entry point of the bot application.
- `.env` - Environment variables file (excluded from the repository for security).

## Getting Started

1. Clone the repository.
2. Create a `.env` file in the root directory with your environment-specific variables.
3. Install dependencies:

```bash
npm install
```

4. Run the bot:

```bash
node app.js
```

## Important

- The `.env` file and `json.sqlite` database file are excluded from the repository to protect your personal and sensitive information.
- Make sure to properly configure your `.env` file before running the bot.
